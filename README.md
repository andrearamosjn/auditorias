# auditorias
Auditoría realizada el 12 de febrero de 2026 sobre la versión web de claude.ai (plan gratuito, idioma español, navegador Chrome 144 en macOS). Se capturaron 12 pantallas mediante Playwright con inspección manual complementaria. Se extrajeron todos los design tokens, componentes, iconos y patrones de interacción visibles en la interfaz pública. Los ratios de contraste se calcularon según WCAG 2.1 AA.
